# Home Landing Specification (PRD Compatible)

적용 대상

```
apps/web/src/app/(store)/page.tsx
```

Rendering

```
ISR
revalidate: 300
```

데이터 소스

```
Firestore
cms/home
books
```

PRD Section 19 규칙 유지.

---

# 1. UX 흐름

온라인북 PRD의 핵심 UX

```
감성 → 탐색 → 발견 → 구매
```



Home 페이지는 이 흐름을 그대로 따른다.

---

# 2. 페이지 구조

최종 구조

```
Header

Search

Hero (서점 감성)

Quick Navigation

Bestseller

Featured Curation

Monthly Pick

New Books

Category Explore

Theme Curation

Events

Content

About Bookstore

Footer
```

이 구조는 **온라인북 PRD 랜딩 UX 그대로 유지하면서**
**기존 기술 PRD와 충돌 없이 구현 가능하도록 제한한 것**이다.
